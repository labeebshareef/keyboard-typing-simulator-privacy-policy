Privacy Policy

Data Collection

This Chrome extension collects anonymous usage analytics through Google Analytics 4, including:

  • Button clicks and feature usage within the extension
  • Settings preferences and configuration changes
  • Number of input fields detected on webpages
  • Typing session statistics (field count, text length)

No Personal Data

We do not collect, store, or transmit any personally identifiable information, typed content, or sensitive user data. All typing simulation occurs locally on your device.

Data Usage

Analytics data is used solely to improve extension functionality and user experience. No data is sold or shared with third parties beyond Google Analytics.

Data Storage

Anonymous usage data is stored by Google Analytics using localStorage on your device for session tracking. No user content or personal information is stored.

User Control

You can disable analytics tracking by turning off extension permissions in Chrome settings. The extension will continue to function normally without analytics.
